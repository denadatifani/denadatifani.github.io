Perpustakaan Politeknik Negeri Medan

Berikut ini adalah gambar perpustakaan polmed:

Perpustakaan Politeknik Negeri Medan (POLMED) mempunyai tujuan menyediakan informasi yang berkaitan dengan sarana pendukung dalam melaksanakan Tri Dharma Perguruan Tinggi dengan cara menghimpun, memilih, mengolah, merawat serta melayani sumber informasi kepada lembaga induk khususnya dan masyarakat akademis pada umumnya.

Perpustakaan sebagai bagian integral dari Politeknik Negeri Medan , bersama-sama dengan jurusan dan unit lainnya di lingkungan Polmed melaksanakan tridharma perguruan tinggi. Perpustakaan Politeknik Negeri Medan berdiri pada tahun 1983, satu tahun setelah berdirinya Politeknik.

Berlokasi di Jl. Almamater No.1, Padang Bulan, Kecamatan Medan Baru, Kota Medan.

KELEBIHAN DAN KEKURANGAN HTML5 DENGAN HTML

HTML5 memiliki banyak kelebihan dibandingkan dengan HTML versi sebelumnya. Di antara kelebihan-kelebihan tersebut adalah:

Kelebihan HTML5:

1. Dukungan untuk Multimedia: HTML5 memiliki tag-tag seperti video dan audio yang memungkinkan penggunaan multimedia tanpa memerlukan plugin tambahan seperti Flash.

2. Canvas: Element canvas memungkinkan pengembang untuk menggambar grafis, animasi, dan visualisasi kompleks secara dinamis di dalam browser.

3. Local Storage: HTML5 mendukung penyimpanan lokal (local storage) yang memungkinkan aplikasi web menyimpan data di sisi klien dengan lebih efisien.

4. Dukungan untuk Mobile: HTML5 memiliki fitur-fitur yang mendukung pengembangan aplikasi web responsif yang dapat diakses dengan baik di perangkat mobile.

5. Dukungan untuk Geolocation: Memungkinkan aplikasi web untuk mengakses lokasi pengguna melalui JavaScript.

6. Semantik yang Lebih Baik: HTML5 memperkenalkan elemen-elemen semantik seperti header, footer, nav, dan lainnya, yang membantu mesin pencari dan pengguna memahami struktur halaman dengan lebih baik.

Kekurangan HTML5:

1. Dukungan Peramban Lama: Beberapa peramban lama mungkin tidak sepenuhnya mendukung semua fitur HTML5, yang dapat menyebabkan masalah kompatibilitas.

2. Kompleksitas: Beberapa fitur HTML5, seperti Canvas atau WebSockets, dapat memerlukan pengetahuan yang lebih mendalam dalam pemrograman web.

3. Keamanan: Meskipun HTML5 memiliki beberapa peningkatan dalam hal keamanan, ada juga potensi masalah keamanan jika tidak digunakan dengan benar.

4. Pemrosesan yang Lebih Berat: Beberapa fitur HTML5 dapat memerlukan lebih banyak daya pemrosesan, terutama pada perangkat dengan kinerja rendah.

5. Ketergantungan pada JavaScript: Banyak fitur HTML5 memerlukan JavaScript untuk berfungsi sepenuhnya, yang dapat menjadi masalah jika pengguna menonaktifkan JavaScript di peramban mereka.

Pengertian HTML dan CSS

Pengertian HTML dan CSS

HTML adalah bahasa markup standar yang digunakan untuk membuat struktur dan konten halaman web. Dalam HTML, pengembang web mendefinisikan elemen-elemen seperti teks, gambar, tautan, formulir, dan elemen lainnya yang membentuk tampilan halaman web. HTML menggunakan tag-tag untuk menandai elemen-elemen ini sehingga peramban web tahu cara menampilkan mereka.

CSS adalah bahasa stylesheet yang digunakan untuk mengontrol tata letak dan tampilan halaman web. Dengan CSS, Anda dapat mengatur warna, ukuran, spasi, jenis font, dan banyak properti tata letak lainnya untuk elemen-elemen HTML. Penggunaan CSS memungkinkan pemisahan antara struktur dan tata letak, sehingga HTML dapat fokus pada struktur konten, sementara CSS mengontrol cara konten tersebut ditampilkan.
